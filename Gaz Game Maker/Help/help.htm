Getting Started

Writing games is very hard but with the help of gaz game maker you can make writing games a lot easier. Gaz Game Maker
takes care of all the hard coding like collision animated graphics, sounds, objects, and rooms, and gives you an easy to use
graphic user interface for designing your games. You will still need some programming skills to write your games in Gaz Game Maker.

The settings window

The debug collision setting if checked will draw two rectangle for all object with a sprite
the frist rectangle is draw in red and will be the same size as the objects sprite the
secoend rectangel is draw in white and it size is equal to the bounding box settings
this is not ment for the public.

The esc key ends the game setting if checked will end the game when the key esape is pressed on the keyboard.

The debug caption setting if checked will show all sorts of information in the windows tilte bar
this is not ment for the public.

The show cursor setting if checked will show the cursor when the mouse is in the game window if its not checked the mouse cursor will not be show.

The show caption in full screen setting if checked will draw the caption for the room.

The draw score setting if checked will add the score to the room caption.

The draw health setting if checked will add the health to the room caption.

The draw lives setting if checked will add the lives to the room caption.

The can pause setting if checked will let the game be pause by pressing the f5 key on the keyboard.

in game shortcut keys
f1 show the game help
f2 debug game
f3 screen shoot
f4 full screen or normal screen
f5 pause game
f6 save the game
f7 loads the game
